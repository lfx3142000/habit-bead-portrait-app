# Habit Beads Portrait Roadmap Options

This document captures candidate next steps for the portrait app. These are planning options only, not committed implementation decisions.

## Table and tracker polish

- Add smooth vertical scrolling inside the tracker area so users can add many habits without pushing controls off screen.
- Keep horizontal and vertical table scrolling predictable without sticky headers or sticky columns.
- Add subtle scroll indicators only if phone testing shows users need them.
- Consider a stronger current-day anchor so portrait opens near today while still allowing review of older days.

## Widget options

### Today widget

- Show today's selected habits as compact bead rows.
- Tap a habit to add one bead.
- Open the app for edit, decrement, or detailed history.
- Recommended first widget because it supports the daily logging loop.

### Weekly strip widget

- Show a 7-day mini grid for a small set of selected habits.
- Use theme-aware colors and simple bead states.
- Tap the widget to open the full tracker focused near today.

### Single habit widget

- Show one high-priority habit with today's count or bead state.
- Include a quick add action.
- Useful for habits such as water, medication, stretching, reading, or prayer.

### Reminder path

- Add an optional gentle reminder notification after widgets are stable.
- Keep reminders quiet and user-controlled.
- Avoid nagging patterns that make the app feel stressful.

## Monetization options

### Recommended model

- Free app with the core daily habit loop intact.
- One-time Pro upgrade for richer customization and power-user features.
- Avoid subscriptions unless cloud sync, AI coaching, or ongoing server-backed features are added.

### Free tier candidates

- Local-first tracking.
- A limited but useful number of habits.
- Core portrait and landscape tracker.
- Basic themes.
- Basic today widget.

### Pro tier candidates

- Unlimited habits.
- More widgets and widget customization.
- Custom themes, calmer palette packs, and habit color sets.
- Habit icons or initials.
- Export and backup.
- Optional app lock or privacy mode.

### Pricing ideas

- Pro one-time upgrade around $4.99.
- Optional supporter upgrade around $9.99 for extra themes or to support development.

## Product principles

- Do not paywall the core daily logging loop.
- Keep the app local-first unless cloud features are intentionally added.
- Make Pro feel like an upgrade for users who already trust the app, not a restriction.
- Prefer calm, low-pressure habit support over aggressive streak or reminder mechanics.
