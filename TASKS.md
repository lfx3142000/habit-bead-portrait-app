# Habit Bead App — Task List

## Project Summary

Android mobile habit tracker called Habit Beads. Table-based interface with last ~2 weeks as columns and user-defined habits as rows. Tap cells to log completion (up to 9 times). Persistent storage, landscape layout, gamified bead visualization. Built in Java with Android Studio.

## App Specification

- Name: Habit Beads
- Platform: Android (Java, Android Studio)
- Layout: Locked landscape
- Table: Col 1 = habit names (25% width, editable), Row 1 = day/date headers
- Cells: click count 1-9, scrollable with frozen header
- Storage: XML preferences file
- Toolbar: App name + 4 action buttons
